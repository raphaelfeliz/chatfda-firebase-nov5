# **App Name**: Door and Window Configurator

## Core Features:

- Question Navigation: Navigate through a series of questions to determine the user's needs for doors and windows based on the provided triage.js file.
- Dynamic Option Display: Dynamically render options (images and labels) for each question, using the questionState provided by the triage.js file.
- State Management: Use the state management from the triage.js file. It will show updated cards that are presented and updated using the functions of the given Javascript code.
- SKU Retrieval: When a final configuration is reached, display the corresponding SKU, using the given triage.js file.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to convey trust and reliability.
- Background color: Very light grey (#F0F4F8) to provide a clean and neutral backdrop.
- Accent color: Warm orange (#E59866) for interactive elements and highlights, ensuring good contrast.
- Body and headline font: 'Inter' sans-serif, for a clean, modern, neutral style that is readable.
- Use clear and representative icons, focused on doors and windows to visually support each choice.
- Employ a card-based layout with images and clear labels for options to improve the decision-making process.
- Subtle transitions and animations on card selection for user feedback.
